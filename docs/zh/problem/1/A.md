# A. 剧院广场

## 题目描述

贝尔兰首都的剧院广场呈矩形，尺寸为 $n \times m$。在该市周年纪念日之际，人们决定用正方形的花岗岩石板铺设广场。每块石板的尺寸为 $a \times a$。

铺砌广场所需的石板最少数量是多少？石板可以覆盖比剧院广场更大的表面，但必须覆盖广场。不允许打碎石板。石板的边应与广场的某条边平行。

## 输入格式

输入包含三个正整数：$n$、$m$ 和 $a$（$1 \le n,m,a \le 10^9$）。

## 输出格式

输出所需的石板数量。

## 样例

### 输入

```
6 6 4
```

### 输出

```
4

```
